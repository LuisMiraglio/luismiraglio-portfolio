DATACOM
AUTOMATION

Automatización de Configuración de ONUs DATACOM

Este proyecto consiste en una herramienta de automatización diseñada para simplificar y acelerar el proceso de configuración de ONUs (Unidades de Red Óptica) utilizadas en entornos de telecomunicaciones. Utilizando Python y Selenium, desarrollé tres scripts personalizados para los modelos DATACOM DM984-422, DM986-414 y DM986-416 AX30, capaces de realizar toda la configuración inicial del equipo sin intervención manual.

Cada script ejecuta automáticamente tareas clave como:

  • Acceso al panel de administración del módem
  • Configuración de usuario y contraseña PPPoE
  • Asignación de parámetros WAN y VLAN
  • Definición de redes Wi-Fi con nombre, canal, seguridad y potencia
  • Activación o ajuste de funciones como bridge, NAT o firewall

Todo el proceso, que usualmente requiere entre 8 y 15 minutos de configuración manual, se reduce a menos de 2 minutos con mínima interacción del técnico. Esto no solo estandariza las instalaciones, sino que permite escalar procesos de provisión de servicios en empresas de internet (ISPs) con mayor eficiencia.

Diagnóstico Automatizado de ONU en Slots – SecureCRT Script

Este script está diseñado para ejecutarse directamente en SecureCRT, aprovechando su capacidad nativa para correr automatizaciones en entornos de red por consola. La herramienta automatiza el diagnóstico completo de un slot específico en una OLT, simplificando el proceso de monitoreo y resolución de problemas.

La herramienta automatiza las siguientes acciones:

  • Ejecuta el comando onu show para identificar los virtuales disponibles dentro del slot seleccionado
  • Lanza automáticamente 64 consultas onu status, una para cada virtual, consolidando en un solo paso el estado de todas las ONUs asociadas a ese slot
  • Genera una salida completa que muestra estado de conexión (activa, caída, ausente)
  • Reporta valores de potencia óptica (RX y TX)
  • Identifica indicadores de error para diagnóstico rápido

Antes, este análisis debía hacerse manualmente virtual por virtual. Ahora, el script permite diagnosticar el estado completo del slot en segundos, reduciendo drásticamente el tiempo técnico requerido y facilitando tareas preventivas o correctivas. Una solución práctica y eficiente para técnicos de redes y personal de soporte en ISPs que necesitan una visión rápida y detallada del estado de las ONUs en infraestructura FTTH.

Sistema de Control de Stock de Productos

Aplicación web completa desarrollada para gestionar el inventario y distribución de productos, permitiendo el seguimiento detallado del stock disponible y entregas realizadas a técnicos. Esta Single Page Application (SPA) funciona completamente en el lado del cliente, sin necesidad de conexión a internet ni servidores externos.

Funcionalidades principales:

  • Sistema de autenticación para control de acceso seguro
  • Gestión completa de productos (CRUD de transformadores con seguimiento de stock)
  • Administración de técnicos con visualización de productos asignados
  • Registro de entregas con actualización automática del inventario
  • Sistema de respaldo de datos con importación/exportación y respaldos automáticos
  • Base de datos SQL completa ejecutándose directamente en el navegador

El proyecto utiliza una arquitectura MVC simplificada y demuestra cómo implementar una aplicación de gestión empresarial completa con JavaScript vanilla, aprovechando SQL.js para crear una base de datos relacional que funciona enteramente en el navegador. Toda la persistencia de datos se maneja localmente mediante localStorage, lo que permite una experiencia offline completa sin sacrificar la potencia de las consultas SQL.
