Automatización de Configuración de ONUs DATACOM

Problema

Configurar ONUs manualmente tardaba de 8 a 15 minutos por equipo, lo cual relentizaba el flujo de trabajo.

Solución

  • Scripts en Python + Selenium para modelos DM984-422 / DM986-414 / DM986-416 AX30.
  • Automatización completa y sin supervisión: el ejecutable solicita únicamente los datos iniciales (usuario/contraseña del equipo, SSID, password Wi-Fi y nueva contraseña de admin). A partir de ahí realiza toda la secuencia: acceso, aplicación de plantillas, verificación de interfaces y activación de servicios — todo en modo automatizado, sin intervención manual del técnico.
  • La aplicación se distribuye como un ejecutable .exe, de modo que el técnico solo debe iniciarlo y seguir los pasos iniciales, sin configuraciones manuales extra.

Impacto

  • Reducción de tiempo operativo.
  • 0% de errores en la configuración.

Diagnóstico Automatizado de ONUs SecureCRT Script

Este script está diseñado para ejecutarse directamente en SecureCRT, aprovechando su capacidad nativa para correr automatizaciones en entornos de red por consola.

La herramienta automatiza el diagnóstico completo de un slot específico en una OLT, simplificando el proceso de monitoreo y resolución de problemas.

Acciones automatizadas

  • Ejecuta el comando onu show para identificar los virtuales del slot.
  • Lanza automáticamente 64 consultas onu status (una por virtual).
  • Muestra estado de conexión y valores ópticos (RX/TX).
  • Identifica indicadores de error para diagnóstico rápido.

Antes era manual virtual por virtual; ahora el slot completo en segundos.
